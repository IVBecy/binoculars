# Username Lookup

 #### This is a program that looks up, one specific username on many websites.
 ### Preview
 ![Preview](uname.png)
 
 ### Usage: ```spy.py [username]```
 
 ## Modules:
Just type into your console `pip install -r requirements.txt`
