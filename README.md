# Binoculars
Username lookup accross social networks

## Usage: 
```
python bino.py --help

        Usage: bino.py  -u [username] -f [file name]

        REQUIRED:
        -u or --username

        OPTIONAL:
        -f or --file:           Output file
        -v or --verbose:        Verbose output
```
 
### Dependencies:
```
pip install -r requirements.txt
```
